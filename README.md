#### YOLO给出框框之后,SAM分割,解决影子和物体无法区分的问题
利用SAM模型本身自带的负点优化,想办法进行负点采样(在影子区域采点,进行进一步区分)环节加一个「亮度 / 梯度筛选」：
- 正点锚定本体,同时根据本体亮度来确定亮暗区分阈值,负点区分阴影
- 阴影一般出现在框框的边缘,所以只在边缘进行负点采样(既可以加快计算,又可以避免物体本体被负采样)
- 不全局检测阴影，只对「框内边缘候选负点」做 1 秒判断：如果这个点比物体中心暗很多 → 判定为阴影 / 背景，作为负点；如果亮度接近 → 判定为物体本体，跳过这个点
本质：只在边缘区域里挑「暗的点」当负点，亮的点（物体本体）不做负点，避免误切连接的物体边缘
